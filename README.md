# Hi, I'm Theo 👋

I'm a Computer Science student and software engineer who loves building scalable, real-world products. I specialize in serverless infrastructure, data-heavy applications, and utilizing agentic AI workflows to accelerate the engineering lifecycle.

### 🛠️ Core Stack & Tools
* **Frontend:** React, Next.js
* **Backend & Infrastructure:** Node.js, Supabase, Cloudflare R2, Serverless GPUs (Modal)
* **Automation:** GitHub Actions, Agentic AI Tooling (Claude Code, Lovable)

---

### 🚀 Featured Projects

#### [GuestMatch]([Link to Repo](https://github.com/TheoTDM/GuestMatch))
An AI-driven event photography platform that automates photo distribution using facial recognition. 
* **Architecture:** Built with Next.js and Supabase, utilizing Cloudflare R2 for object storage.
* **Compute:** Leveraged Modal's serverless GPUs for high-performance facial recognition inference, optimizing against cold-start latency to maintain a snappy user experience.

#### [Social Media Auto-Publisher]([Link to Repo](https://github.com/Futurelabs-app/monorepo))
A comprehensive automation engine for creating, scheduling and publishing content. *(1st Place, Startup Studio course)*
* **Mechanics:** Orchestrated multimodal payloads (handling both complex image and video processing).
* **Automation:** Engineered the scheduling layer natively using GitHub Actions for reliable, cron-based execution.

#### [Calma - IoT Meditation Device](Link to Repo)
A hardware-meets-software project prototyping an IoT heating blanket for body-scan meditation.
* **Details:** Spent a year prototyping and testing the physical device and software bridge, demonstrating end-to-end product lifecycle management.
